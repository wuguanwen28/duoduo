# duoduo 让你的小猫陪你上班摸鱼

- 一只住在桌面右下角的小猫咪。透明、置顶、不挡操作，会自己发呆睡觉，眼神跟着你的鼠标转，还能跟你互动。

## 它会做什么

- 🐾 **自己也会动**：发呆、睡觉，时不时眨眼、摇尾巴、动耳朵、等你投喂
- 👀 **眼神跟随**：鼠标移动时盯着你看，凑近猫头时会正视前方
- 🖱️ **多种互动**：单击逗猫、双击最小化、右键打开菜单、长按、拖动搬家
- 💬 **会说话**：可以给它配几句台词，它会随机冒泡念叨
- 🪟 **不碍事**：透明窗口、始终置顶、不占任务栏；空白区点击直接穿透到下面的程序
- 🖼️ **多显示器**：换屏也不跑丢，猫咪不会飞出屏幕

## 📥 下载

到 [Releases 页面](https://github.com/wuguanwen28/duoduo/releases) 下载最新版，解压后运行 `duoduo.exe` 就行，免安装。

> 第一次运行需要给它准备猫咪素材（见下面「换一只猫」），否则会显示引导提示。

## 🎨 玩法

### 跟它互动

在猫咪身上：

- **左键单击** —— 逗它一下（随机小动作）
- **双击** —— 藏起来（最小化到托盘）
- **右键** —— 打开猫爪菜单（大小、透明度、投喂、睡觉、设置等）
- **拖动** —— 把它搬到屏幕任意位置

托盘图标右键也能打开设置或退出。

### 换一只猫

猫咪的每一帧画面都放在程序旁边的 `resources/` 文件夹里，**换猫、加动作只要替换图片，不用重新安装或编译**。

最简单的玩法：找一套别人做好的猫咪素材，整个替换掉 `resources/` 即可。

想自己做的话，打开「设置 → 资源」有可视化编辑器，可以管理跟随、动作、行为，保存后即时生效。设置里还内置了一个「视频转 WebP」工具——导入猫咪视频，自动抽帧 + 绿幕抠图，直接变成可用的素材。

### 自定义互动方式

「设置 → 显示与交互」里可以：

- 给左键 / 双击 / 右键 / 长按分别绑定不同动作
- 配置全局快捷键（比如一键唤出、一键投喂）
- 编辑右键菜单的 5 个选项
- 调整大小、透明度、是否置顶、是否点击穿透

## ❓ 常见问题

**为什么打开是空白 / 没有猫？**
第一次运行需要准备 `resources/` 素材文件夹。放好猫咪帧图片和 `manifest.json` 后重新打开即可。

**怎么让它不挡我点东西？**
右键菜单开启「点击穿透」，猫咪就完全透明了；按住 Ctrl 可以临时恢复操作。

**它会自动更新吗？**
会。有新版本时设置里会提示，确认后自动下载替换。
