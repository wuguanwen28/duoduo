# duoduo 让你的小猫陪你上班摸鱼

- 一只住在桌面右下角的小猫咪。透明、置顶、不挡操作，会自己发呆睡觉，眼神跟着你的鼠标转，还能跟你互动。

## 它会做什么

- 🐾 **自己也会动**：发呆、睡觉，时不时眨眼、摇尾巴、动耳朵、等你投喂
- 👀 **眼神跟随**：鼠标移动时盯着你看，凑近猫头时会正视前方
- 🖱️ **多种互动**：单击逗猫、双击最小化、右键打开菜单、长按、拖动搬家
- 💬 **会说话**：可以给它配几句台词，它会随机冒泡念叨
- 🪟 **不碍事**：透明窗口、始终置顶、不占任务栏；空白区点击直接穿透到下面的程序

## 自定义小猫

在设置页面中，点击「资源」选项卡，可以查看视频生成教程。
生成好的视频可以使用「视频转 WebP 图片」工具，直接变成可用的素材。


## ❓ 常见问题

**为什么打开是空白 / 没有猫？**
第一次运行需要准备[默认素材](https://gitee.com/wuguanwen28/duoduo/releases/download/v0.1.0/duoduo-resources.zip)文件夹。放好猫咪帧图片和 `manifest.json` 后重新打开即可。

**怎么让它不挡我点东西？**
右键菜单开启「点击穿透」，猫咪就完全透明了；按住 Ctrl 可以临时恢复操作。
