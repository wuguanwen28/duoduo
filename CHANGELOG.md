## [0.1.0] - 2026-06-16

一只透明、置顶、会跟随光标的桌面猫咪宠物。素材完全外置，换猫、加动作都不用重新打包。

### 桌面宠物

- 透明无边框、始终置顶、不占任务栏的桌面窗口，猫咪右下角对齐
- 空白区透明，点击自动穿透到下层程序
- 眼神跟随光标；光标贴近猫头时正视前方
- 自治行为：加权轮换「发呆 / 睡觉」，随机插播眨眼、摇尾、动耳、投喂等小动作
- 单击逗猫（随机动作）、双击最小化、拖动移动
- 多显示器边界约束，猫咪不会跑出屏幕

### 外置资源（核心特性）

- 所有帧放在程序同级 `resources/`，由 `manifest.json` 描述
- 换猫 / 加动作只改文件夹，无需重新编译
- 可指定默认行为；缺素材时显示引导与教程

### 设置与交互

- 右键菜单：大小、透明度、偷看开关、点击穿透、校准猫头、睡觉、投喂、老板来了、下班
- 头部校准，设置自动记忆
- 系统托盘：左键最小化/恢复，右键 设置 / 退出

### 可视化资源编辑器

- 托盘「设置」打开独立窗口，可视化编辑 跟随 / 动作库 / 行为库
- 支持选目录、测试播放、时间单位（秒/分/时）、默认行为、动作与行为中文名
- 保存后自动热重载，改动即时生效

### 打包与发布

- `pnpm app:build` 自动产出绿色版 zip（exe + resources）
- 推 `v*` 标签触发 GitHub Actions 自动发布 Release
