## 📦 下载说明

### windows
`windows 会把未知来源的程序标记为危险程序，忽略即可`

| 文件                              | 内容                                    |
| --------------------------------- | --------------------------------------- |
| **`duoduo.exe`**                  | 主程序，更新时覆盖原 `duoduo.exe` 即可  |
| **`duoduo-resources.zip`**        | 默认素材 （`resources/`）               |
| **`duoduo-__VERSION__-full.zip`** | 完整包：主程序 + 默认素材，下载解压即用 |
