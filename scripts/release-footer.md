## 📦 下载说明

### windows

> 程序未做数字签名，Windows 会弹「Windows 已保护你的电脑 / 未知发布者」，这是正常提示，可放心运行：
>
> 1. 在蓝色弹窗点「**更多信息**」；
> 2. 再点下方出现的「**仍要运行**」即可。
>
> 若右键 `duoduo.exe` → 属性，底部有「解除锁定（Unblock）」，勾选并确定后可不再提示。

| 文件                              | 内容                                    |
| --------------------------------- | --------------------------------------- |
| **`duoduo.exe`**                  | 主程序，更新时覆盖原 `duoduo.exe` 即可  |
| **`duoduo-resources.zip`**        | 默认素材 （`resources/`）               |
| **`duoduo-__VERSION__-full.zip`** | 完整包：主程序 + 默认素材，下载解压即用 |
